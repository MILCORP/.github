## 🚀 Getting Access

1. **Join the organization**
   - Team members should accept the invite sent from [noreply@github.com](https://www.office365.us/search?q=noreply%40github.com&EntityRepresentationId=9697b7c6-2f61-457e-a67e-360323834a19) to their MIL email inbox. If you can’t find it, check spam or request a resend from the org admins. [3](https://www.office365.us/search?q=noreply%40github.com)

2. **Request repo permissions**
   - Ask to be added to the appropriate team(s) (e.g., **AI Team Lab**) with *Write* access for development and *Read* for review-only.

3. **MIL network considerations**
   - Our demos and integrations are deployed using **Azure OpenAI (GPT‑4.1)** within the MIL environment to align with client realities and approvals. If you need service principals, key vaults, or pipelines, contact the AI Lab leads.
